#  Crochet Procurement Classification Project

## Overview
This project analyzes crochet spendings using **MySQL 8.0** and **Python**, focusing on yarn suppliers **Alize Burkum** and **YarnArt**.  
It demonstrates skills in database design, SQL analytics, and Python visualization.
